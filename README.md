# 📦 Sistema de Controle de Estoque

API REST desenvolvida com Python e FastAPI para gerenciamento de produtos e movimentações de estoque.

Deploy: https://controle-de-estoque-udzs.onrender.com
<img width="1856" height="952" alt="image" src="https://github.com/user-attachments/assets/e9c35cb2-aba3-48a4-8431-a0c47b74976c" />


## 🚀 Funcionalidades

* Autenticação de usuários com JWT
* Cadastro, edição e exclusão de produtos
* Controle de entrada de estoque
* Controle de saída de estoque
* Consulta de produtos cadastrados
* Validação de dados com Pydantic
* Documentação automática com Swagger

## 🛠 Tecnologias Utilizadas

* Python
* FastAPI
* SQLAlchemy
* PostgreSQL
* Pydantic
* JWT Authentication
* Pytest
* Git e GitHub

## 🎯 Objetivo

Este projeto foi desenvolvido para praticar conceitos fundamentais de desenvolvimento backend, incluindo:

* APIs REST
* Autenticação e autorização
* Banco de dados relacionais
* Arquitetura em camadas
* Regras de negócio
* Boas práticas de desenvolvimento
